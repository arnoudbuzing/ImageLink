# ImageLinkGrayscale

`ImageLinkGrayscale[input, output]` converts an image file to grayscale and saves the result.

## Parameters

*   **input** (String): The file path to the input image.
*   **output** (String): The file path where the grayscale image will be saved.

## Examples

### Convert to Grayscale

```wolfram
ImageLinkGrayscale["photo.jpg", "photo_gray.jpg"]
```
