# ImageLinkInvert

`ImageLinkInvert[input, output]` inverts the colors of an image file and saves the result.

## Parameters

*   **input** (String): The file path to the input image.
*   **output** (String): The file path where the inverted image will be saved.

## Examples

### Invert Colors

```wolfram
ImageLinkInvert["logo.png", "logo_inverted.png"]
```
